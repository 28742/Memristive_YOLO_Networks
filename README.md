# Memristive YOLO Networks

A research project exploring the integration of memristive devices into YOLO (You Only Look Once) object detection networks for neuromorphic computing applications.

## Overview

This repository contains code and experiments for implementing and evaluating YOLO-based object detection networks that incorporate memristive (memristor-based) components. The goal is to investigate how memristive devices can be used to accelerate neural network inference and reduce power consumption in edge computing scenarios.
